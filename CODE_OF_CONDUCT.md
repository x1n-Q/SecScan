# Code of Conduct

## Our Standard

We want this project to be constructive, respectful, and safe for contributors of all experience levels.

## Expected Behavior

- be respectful and professional
- assume good intent
- give actionable review feedback
- keep technical disagreements focused on the work
- help keep the project defensive and responsible

## Unacceptable Behavior

- harassment or discrimination
- abusive or insulting language
- publishing private information without permission
- encouraging misuse of the project for unauthorized access or harm

## Enforcement

Project maintainers may remove comments, issues, pull requests, or contributions that violate these expectations.

## Scope

This code of conduct applies to repository discussions, issues, pull requests, and other project spaces.
